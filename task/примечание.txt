1. Accets - папка с файлами, где лежат прилинкованные headers, footers, popups для всех разрешений экрана;

2. Img - папка с картинками, которые используются для сайта;

3. Psd - готовые макеты сайта для разрешений 320px, 940px, 1170px.
   В папке Png лежат скрины для 1170px и 320px;

4. Sitemap - карта сайтa;

5. UI_kit - собраны все элементы сайта и показаны их состояния.
